### 🌟 Next.js 프로젝트 생성하기

```
npx create-next-app@latest [프로젝트명]
```

- [Reference](https://nextjs.org/docs/getting-started/installation)

### 🌟 Next.js Route System 실습하기

- init한 프로젝트에 라우트 만들기

```
- "/" (home)
- "/posts"
```

- posts route에는 https://jsonplaceholder.typicode.com/를 활용하여 ssr을 구현한 게시글 목록을 보여주세요
