## 👩‍🚀 네트워크 프로토콜 표준화

### 🚀 네트워크 프로토콜

- 다른 장치들끼리 데이터를 주고받기 위해 설정된 공통된 인터페이스

```
🌝 이는 기업이나 개인이 발표해서 정하는 것이 아닙니다!
```

- 표준화 단체에 의해 정해진다. (IEEE, IETF)

> `IEEE802.3`

- 유선으로 LAN 구축 시 사용되는 프로토콜
- 만든 기업이 다른 장치라도 서로 데이터를 수신할 수 있게 한다.
- 예: **_HTTP_**
  - 서로 약속된 인터페이스
  - 이 프로토콜을 통해 노드들은 웹 서비스를 기반으로 데이터를 주고받을 수 있다.
