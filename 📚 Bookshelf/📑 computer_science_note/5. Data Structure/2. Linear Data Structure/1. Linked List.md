## 🧙‍♀️ 연결 리스트

- 데이터를 감싼 노드를 포인터로 연결해서 공간적인 효율성을 극대화시킨 자료 구조
- prev 포인터와 next 포인터로 앞과 뒤의 노드를 연결시킨 것
- 맨 앞에 있는 노드를 `head`라고 한다.
- 시간 복잡도
  |삽입|삭제|탐색|
  |:--:|:--:|:--:|
  |O(1)|O(1)|O(n)|
